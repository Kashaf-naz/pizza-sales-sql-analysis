This project delivers **end-to-end sales analytics** for a pizza business using SQL.  
It uncovers trends in orders, revenue, product performance, customer behavior patterns, and category contributions.

---

## ğŸ” **Key Analyses Performed**

- **Total orders & total revenue**  
- **Top-selling pizzas** & most common pizza size  
- **Revenue contribution by category**  
- **Hourly vs daily ordering behavior**  
- **Monthly cumulative revenue** using **CTEs + Window Functions**  
- **Revenue ranking** (RANK, DENSE_RANK)  
- **Top 3 pizzas per category**  
- **Least-selling pizzas** for business improvement
